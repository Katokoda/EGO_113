# EGO_113
[Esperanto](https://fr.wikipedia.org/wiki/Esp%C3%A9ranto) translation of captions from [a french video](https://www.youtube.com/watch?v=j2JDvXvymBQ) made by EGO

## En français
1. Ce depôt a comme but de traduire les sous-titres d'[une récente vidéo](https://www.youtube.com/watch?v=j2JDvXvymBQ) d'EGO en Espéranto.

2. **Qu'est l'Espéranto ?**\
L'Espéranto est [une langue construite intérnationale](https://fr.wikipedia.org/wiki/Esp%C3%A9ranto). ayant comme but original l'inter-compréhension des peuples.

3. **Pourquoi cette vidéo devrait-elle être traduite en Espéranto ?**\
Cette vidéo a comme thème la compréhension des autres *formes de vies* et comme but d'apprendre à les respecter. L'Espéranto a comme but de faciliter la compréhension entre les humains. Le lien est symboliquement fort.

4. **Comment participer ?**\
Contactez Katokoda sur GitHub, sur Discord (katokoda), sur Telegram (@Samuel_Bel) ou par e-mail (katokoda@gmail.com).
Vous pouvez aussi faire des Pull-Requests sur ce depôt.

5. **Est-ce que les sous-titres seront disponibles une fois la vidéo traduite?**\
Seul EGO peut ajouter des sous-titres sa vidéo. Une fois que le projet aura fait ses premiers pas, nous contacterons EGO pour savoir si ce projet à un sens.

6. **Quels sont les autres fichiers de ce depôt ?**\
    - Dans un premier temps, la conceptothèque nous servira de liste de concepts fréquemment utilisés dans la vidéo et/ou qui ne sont pas fréquents en Espéranto. Ce document servira de référence quand nous pourrons traduire la suite.
    - Dans un second temps, nous aurons accès au fichier de sous-titrage français de la vidéo et pourront le traduire vers l'Espéranto.


## Esperante
1. Tiu deponejo havas celon traduki subtitolojn de [nova filmeto](https://www.youtube.com/watch?v=j2JDvXvymBQ) de EGO, franca Youtube-isto, en Esperanto.

2. **Kio estas Esperanto?**\
Mi pensas ke, se vi tion komprenas, vi konas Esperanton jam ;)

3. **Kial traduki tiun filmeton Esperante?**\
Tiu filmeto temas pri kompreno de *aliaj viv-estaĵoj* kaj celas inciti respekton. Esperanto celas helpi homojn sin interkompreni. La ligo estas simbole forta.

4. **Kiel partopreni?**\
